How Do Classes Work

Ruby Classes

November 29, 2014

This is how you create a simple class:

class Books
def initialize(title, author)
@title = title
@author = author
end

def cool_books
"I love #{title}, by #{author}!"
end
end
This is how you create an object for the class and call its methods:

Books.new("The Well-Grounded Rubyist", "David Blake").cool_books
The main reason we use instance variables is to be able to acces those variables throughout the entire instance. For example, we created an instance of the Book class with the two arguments, "The Well-Grounded Rubyist" and "David Blake".These two parameters were passed to the initialize method at the beginning of the class, but were also accessed afterwards in the following cool_books method. If the title and author variables were local variables we were not able to refer to them in the next method.
